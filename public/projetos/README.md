# Projetos publicados

Cada projeto criado pelo Visualizador recebe uma pasta exclusiva:

```text
public/projetos/nome-do-projeto/
├── index.html    # página publicada
├── source.txt    # código-fonte original usado pelo editor
└── project.json  # nome, descrição, tecnologias e links do cartão
```

Ao atualizar um projeto, somente esses três arquivos da pasta correspondente são alterados.
As demais pastas não são regravadas.
